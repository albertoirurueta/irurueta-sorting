= Installation
:description: Add irurueta-sorting to a Maven or Gradle project.

Irurueta Sorting is a Maven artifact that can be integrated into Java projects using group id
`com.irurueta` and artifact id `irurueta-sorting`. The project is built as a Java 21 Maven JAR.

== Maven configuration

Add this dependency to your `pom.xml`:

Latest release:

[source,xml]
----
<dependency>
    <groupId>com.irurueta</groupId>
    <artifactId>irurueta-sorting</artifactId>
    <version>1.4.0</version>
    <scope>compile</scope>
</dependency>
----

Latest snapshot:

[source,xml]
----
<dependency>
    <groupId>com.irurueta</groupId>
    <artifactId>irurueta-sorting</artifactId>
    <version>1.5.0-SNAPSHOT</version>
    <scope>compile</scope>
</dependency>
----

== Gradle configuration

Include this in your `build.gradle.kts`:

[source,kotlin]
----
dependencies {
    implementation("com.irurueta:irurueta-sorting:1.4.0")
}
----

== Package imports

Runtime types are located in `com.irurueta.sorting`. Common imports include:

[source,java]
----
import com.irurueta.sorting.Sorter;
import com.irurueta.sorting.SortingMethod;
import com.irurueta.sorting.SortingException;
----
